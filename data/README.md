# Données CIQUAL

valeurs pour 100 g (énergie, protéines, glucides, lipides, fibres, sucres, sel).

Source : **Anses. 2020. Table Ciqual de composition nutritionnelle des aliments.**
Réutilisation libre avec mention obligatoire de la source.
